Twitter. Canvas. 10K.

Press the spacebar to toggle this screen.

This is an experiment to see how HTML5 canvas can visualise data in interesting ways. Each of the 100 points you see is a Twitter user. You're looking at the people I follow by default (@robhawkes), but you can change that below. Click on a point and you'll see more information about that user. You can also sort the users in different ways. All this in under 10KB of code. Awesome!

If nothing happens the Twitter API is probably down or the API limit has been reached. If so, try again later. Some things are out of my control unfortunately.
